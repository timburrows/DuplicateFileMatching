# Duplicate File Matching

### Table of Contents
1. [Prerequisites](#1-prerequisites)
2. [Usage](#2-usage)

## 1. Prerequisites
- .NET Core 3.1
- IDE or a terminal emulator to run the CLI.

## 2. Usage

Terminal:
```
$ dotnet restore
$ cd DuplicateFileMatching.ConsoleApp/
$ dotnet run
```

IDE:

Build and Run solution in Visual Studio, or JetBrains Rider (tested).
